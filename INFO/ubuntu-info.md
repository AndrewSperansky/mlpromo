### Что делает команда chmod +x scripts/smoke_test.sh?
chmod (от change mode) — команда в Linux/Unix‑системах для изменения прав доступа к файлам и директориям.

- Флаг +x означает:
- добавить право на исполнение (execute) для файла;  
- после этого скрипт можно запускать как программу.  

Разбор команды:

>bash
chmod +x scripts/smoke_test.sh
chmod — сама команда;

+x — добавить право исполнения;


## jq — это утилита для обработки JSON в командной строке. Позволяет:

- извлекать данные из JSON;
- фильтровать, преобразовывать и форматировать JSON‑объекты;
- автоматизировать работу с API, конфигурационными файлами и логами.

### Пример использования:

Допустим, есть файл data.json с содержимым:

>json
{
  "name": "Alice",
  "age": 30,
  "hobbies": ["reading", "swimming"]
}

- Извлечь имя:  

>bash  
jq '.name' data.json  
Вывод: "Alice"

- Получить возраст:  

>bash  
jq '.age' data.json  
Вывод: 30  

- Отфильтровать хобби длиннее 6 символов:  

>bash  
jq '.hobbies[] | select(length > 6)' data.json  
Вывод: "swimming"

`mkdir -p ./models`

Что делает:  
Создаёт директорию models в текущей рабочей папке (./).  
Ключ -p гарантирует:  
если папка уже существует — ошибки не будет;  
если нужны промежуточные папки — они создадутся автоматически.


`cp model_stage2.cbm ./models/model.cbm`

Что делает:  
Копирует файл model_stage2.cbm из текущего каталога в папку ./models/.  
При копировании переименовывает файл в model.cbm.  
./models/ — целевая папка;  
model.cbm — новое имя файла.
