## 2. Бизнес-назначение модели

Модель предназначена для:
- прогнозирования эффекта промо-акций
- поддержки решений «делать / не делать промо»
- оценки ожидаемого uplift продаж

Модель используется как **рекомендательная**, а не как единственный источник принятия решений.

---

## 3. Область применения и ограничения

### Применимо:
- массовые SKU
- стандартные промо-механики
- исторические паттерны продаж

### Не применимо:
- новые SKU без истории
- нетипичные акции (разовые, ручные)
- экстремальные скидки вне обучающего диапазона

---

## 4. Используемые признаки (features)

| Feature | Описание |
|------|--------|
| price | Цена SKU |
| discount | Размер скидки (%) |
| avg_sales_7d | Средние продажи за 7 дней |
| promo_days_left | Дней до окончания промо |
| stock_level | Остаток на складе |

Список признаков является **контрактом** и контролируется через feature schema.

---

## 5. Обучающие данные

- Источник: корпоративное хранилище
- Период: исторические данные промо-акций
- Тип данных: табличные, агрегированные

Предобработка:
- удаление выбросов
- заполнение пропусков
- нормализация не применяется (CatBoost)

## 7. Explainability (SHAP)

Для интерпретации модели используется **SHAP (TreeSHAP)**.

Доступные артефакты:
- `shap_summary.json` — глобальная важность признаков
- `shap_values.npy` — значения SHAP на train
- `shap_expected_value.json` — базовый прогноз

SHAP используется:
- для объяснения решений
- для контроля деградации модели
- для аудита

SHAP **не используется** для ручного изменения прогнозов.

---

## 8. Drift Monitoring

Отслеживаются:
- **Data drift** (PSI)
- **SHAP drift** (изменение влияния признаков)

Пороговые значения:
- PSI warning: 0.1
- PSI drift: 0.25
- SHAP change ratio: 30%

Drift используется для:
- рекомендаций по переобучению
- fallback на baseline
- алертинга

---

## 9. Fallback и отказоустойчивость

При критическом drift:
- модель может быть отключена
- используется baseline-прогноз
- бизнес-процессы продолжаются без остановки

Fallback является обязательной частью системы.
